Multi Series Rader Chart
====

Multi Series Rader Chart

## for User
それぞれのデータが同じ3つの指標を持っている場合に利用可能なグラフです。
第1列には二つの指標でユニークになるよう文字列を挿入してください.
You can use this chart only data have  **same** three metrics.


Data is presented by gooスマホ部
